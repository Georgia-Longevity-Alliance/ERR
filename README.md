# ERR — Eliminate, Reprogram, Rebuild

**Grant application:** EIC Pathfinder Challenges 2026
**GitHub:** https://github.com/Georgia-Longevity-Alliance/ERR
**Parent project:** MCARA (Multi-Counter Architecture of Replicative Aging)

## The Hypothesis

Cells degrade old proteins, damaged mitochondria, and worn organelles. The centriole has no disposal pathway. During asymmetric stem cell divisions, the oldest centrioles selectively accumulate.

Three counters of replicative aging can be reset: telomeres, mitochondria, the epigenome. The fourth — the centriole — has never been reset in a somatic cell.

ERR proposes to do exactly that: Eliminate → Reprogram → Rebuild.
